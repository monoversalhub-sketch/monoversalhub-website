## About This Project

This is the public-facing website for Monoversal Hub. It introduces the company, showcases products currently in development, and provides a way for the community to connect, give feedback, and support the work.

The site is built as a single, self-contained HTML file — no frameworks, no dependencies, no build process required.

## Products in Development

### 🆘 SOVR — Community SOS & Emergency Response
A community-powered emergency response platform that connects people in crisis with nearby individuals and organizations who can help — in real time.

- Users raise an SOS from any location
- Nearby responders are notified based on capacity and proximity
- Victims see real-time rescue options and choose the best one
- Three tiers: Individual, Private Organization, Public Organization
- Everyone is both a potential victim and a potential rescuer

**Status:** In Development

---

### 📐 FYTD — AI Measurements & Body Fit Intelligence
An intelligent body measurement companion that stores, analyzes, and makes sense of sizing data — built for tailors, fashion businesses, and individuals.

- Stores and organizes body measurements
- Analyzes data trends and patterns
- Delivers personalized fit recommendations
- Daily tips on measurements, fit, and app usage
- Supports male and female measurement systems

**Status:** In Development

## Deployment

This site is deployed via **Netlify** with automatic publishing from GitHub.

To update the live site:

1. Make your changes to `index.html`
2. Commit and push to the connected GitHub repository
3. Netlify auto-deploys within seconds

To deploy manually on Netlify:
1. Log in to [netlify.com](https://netlify.com)
2. Drag and drop the `index.html` file into your site dashboard
3. Done
